# What is Evangelion?

**Neon Genesis Evangelion** is a renowned Japanese animated TV series created by Hideaki Anno. It blends giant robots with deep psychological themes and symbolic religious imagery.

# Story Basics

The story is set in a post-apocalyptic future after a catastrophic event. Teenagers pilot giant robots called Evangelions to protect Earth from mysterious beings known as Angels.

# Why is Evangelion Special?

- Combines action with profound reflections on life and existence.
- Features characters with complex emotions and psychological struggles.
- Incorporates religious symbolism in unique and thought-provoking ways.

# Key Characters

- **Shinji Ikari**: The main pilot who grapples with fear and his responsibilities.
- Other pilots also face intense personal challenges.

# How Evangelion Changed Anime

Evangelion demonstrated that anime could explore mature, emotional, and philosophical narratives beyond typical action scenes. It has inspired numerous subsequent shows and films.

# Simple Code Example: Modeling an Evangelion Robot in Python

Here’s a brief Python example that models a simple Evangelion robot:

```python
class Evangelion:
    def __init__(self, pilot):
        self.pilot = pilot
        self.status = "Ready"

    def battle(self, enemy):
        print(f"{self.pilot} is fighting {enemy}!")
        self.status = "In battle"


unit01 = Evangelion("Shinji")
unit01.battle("Angel")
```

This example demonstrates basic object-oriented programming by defining an Evangelion unit and simulating a battle.
